1. What is JQuery? 
ANS.- jQuery is a fast, small, and feature-rich JavaScript library. It makes things
      like HTML document traversal and manipulation, event handling, animation, and
      Ajax much simpler with an easy-to-use API that works across a multitude of browsers.



2. How to Apply CSS Using JQuery, How to Add Class and Remove Class in JQuery, 
   JQuery Animation?
ANS.- Syntax:

            $().css(propertyname, value);
            $().css(properties);
     
      addClass() method: It is used to add specified classes to each of the targeted
      elements while animating all style changes.
      Syntax:
            .addClass(className, [duration], [easing], [callback])

      removeClass() method: It is used to remove the classes from the element while animating
      all style changes.
      Syntax:
            .removeClass(className, [duration], [easing], [callback])


3. How to create slider with animation? 
ANS.- Syntax:
             $(".selector").slider(
             { animate: value }
             );
